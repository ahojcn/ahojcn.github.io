JSON：JavaScript Object Notation 的缩写，它是一种数据交换格式。详细资料

JSON -> 字符串：序列化
字符串 -> JSON：反序列化

序列化 stringify

<!DOCTYPE html>
<html lang="en" dir="ltr">
  <head>
    <meta charset="utf-8">
    <title></title>
    <script>
    let json={a: 12, b: 5, c: 'aaa'};    // 注意：这里是不标准的 JSON 写法，但是在 JS 中可以这么写，序列化后，JS 会把它转为标准的 JSON 格式。{"a":12,"b":5,"c":"aaa"}

    console.log(JSON.stringify(json));
    </script>
  </head>
  <body>
  </body>
</html>

反序列化 parse

<!DOCTYPE html>
<html lang="en" dir="ltr">
  <head>
    <meta charset="utf-8">
    <title></title>
    <script>
    let str='{"a":12,"b":5,"c":"aaa"}';
    let json=JSON.parse(str);

    console.log(json);
    </script>
  </head>
  <body>
  </body>
</html>
