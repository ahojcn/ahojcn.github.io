Nodejs-入门
2019-03-27|JavaScript

node 一般是作为 前端和服务器中间的中间层，比如做一些数据处理，数据缓存等，减少服务器压力，也可以处理一些不安全的东西。

用途：

  1. 中间层（安全性、性能、降低主服务器复杂度）
  2. 小型服务
  3. 工具

优势：

  1. 性能高
  2. 利于和前端代码整合

安装

安装node.js https://nodejs.org/

npm：Node Package Manager，一般 npm 比较慢，所以要换源：(注意有些 包 只能用 npm 下载安装，大部分可以用 cnpm 来安装)

npm install cnpm -g --registry=https://registry.npm.taobao.org

卸载

怎么卸载低版本
1.卸载node本身、删除nodejs目录
2.手动删除C:\Program Files\nodejs\node_modules\
3.手动删除C:\users\你\node_modules\

使用

运行 nodejs 程序

使用 node name.js

包

安装/删除包

安装
npm install xxx
npm i xxx

删除
npm uninstall xxx
npm un xxx

cnpm i xxx
cnpm un xx

引入包

const xx=require('xxx');    // xxx 是包名
