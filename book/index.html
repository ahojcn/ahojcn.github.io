问题手册

Mac su

Mac下使用su命令提示sorry，解决办法如下：

➜  Downloads su
Password:
su: Sorry

谷歌之，两个解决方式：

sudo su -  # 输入开始第一个创建用户的密码
sudo -i

或者

sudo passwd root # 同Linux

g++: command not found 解决方法

G++没有装或却没有更新，以下方法都可以试试：

centos：

sudo yum -y update gcc
或者
sudo yum -y install gcc+ gcc-c++

ubuntu：

apt-get update gcc
或者
apt-get install g++

parallels desktop 安装centos7默认密码和root问题

由于系统默认是没有激活root用户的，需要手工进行操作：

sudo passwd
Password：当前的密码
Enter new UNIX password：这个是root的密码
Retype new UNIX password：重复root的密码

使用su和sudo是有区别的，使用su切换用户需要输入所切换到的用户的密码，而使用sudo则是当前用户的密码。
