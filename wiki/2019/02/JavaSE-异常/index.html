异常概述

异常：异常就是Java程序在运行过程中出现的错误。

异常由来：问题也是现实生活中一个具体事务，也可以通过 Java 的类的形式进行描述，并封装成对象。其实就是Java 对不正常情况进行描述后的对象体现。
